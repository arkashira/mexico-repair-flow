# Breakeven Analysis
## Unit Economics
### Cost per Active User
Based on the infrastructure requirements for the mexico-repair-flow product, we estimate the following costs:
- Compute: $0.05 per user per month (assuming 10% utilization of a $5/month server)
- Storage: $0.01 per user per month (assuming 1GB storage per user at $0.01/GB/month)
- Bandwidth: $0.02 per user per month (assuming 100MB bandwidth per user at $0.02/100MB)
Total cost per active user: $0.08 per month

### Pricing Tiers
We propose the following pricing tiers for the mexico-repair-flow product:
1. **Basico**: $19/month (billed annually) - includes quoting and workflow management for up to 10 users
2. **Estándar**: $49/month (billed annually) - includes all Basico features, plus inventory management and reporting for up to 20 users
3. **Empresarial**: $99/month (billed annually) - includes all Estándar features, plus advanced analytics and priority support for up to 50 users

### Customer Acquisition Cost (CAC) Range
Based on industry benchmarks and marketing strategies, we estimate the CAC to be between $50 and $200 per user.

### Lifetime Value (LTV) Estimate
Assuming an average revenue per user (ARPU) of $49/month (mid-tier) and a customer lifetime of 24 months, we estimate the LTV to be:
LTV = ARPU x Customer Lifetime = $49/month x 24 months = $1176

### Break-even Users Count
To calculate the break-even point, we need to consider the CAC and LTV. Assuming a CAC of $125 (mid-range of the estimated CAC range) and an LTV of $1176, we can calculate the break-even point as follows:
Break-even Point = CAC / (LTV - CAC) = $125 / ($1176 - $125) ≈ 0.12 years or approximately 1.4 months
Break-even Users Count = CAC / (Monthly Revenue per User - Monthly Cost per User)
Using the Estándar tier as an example:
Monthly Revenue per User = $49/month / 20 users = $2.45 per user
Monthly Cost per User = $0.08 (as calculated earlier)
Break-even Users Count = $125 / ($2.45 - $0.08) ≈ 53 users

### Path to $10K MRR
To reach $10,000 in monthly recurring revenue (MRR), we can estimate the required number of users for each tier:
1. **Basico**: $19/month × number of users = $10,000
Number of users = $10,000 / $19 ≈ 526 users ( Basico tier, which is unlikely given the low pricing)
2. **Estándar**: $49/month × number of users = $10,000
Number of users = $10,000 / $49 ≈ 204 users (more realistic, considering the mid-tier pricing)
3. **Empresarial**: $99/month × number of users = $10,000
Number of users = $10,000 / $99 ≈ 101 users (most realistic, considering the high-tier pricing and priority support)
A combination of these tiers can also be used to reach the $10K MRR goal. For example:
- 50 users on the Empresarial tier ($99/month) = $4,950
- 100 users on the Estándar tier ($49/month) = $4,900
- 50 users on the Basico tier ($19/month) = $950
Total MRR = $4,950 + $4,900 + $950 = $10,800

This combination would put us slightly above the $10K MRR goal, indicating a viable path forward.
