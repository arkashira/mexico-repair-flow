# Mexico Repair Flow

This project provides a quote generator for a repair shop.

## Usage
1. Create a quote using the `create_quote` method.
2. Get a quote using the `get_quote` method.
3. Update a quote using the `update_quote` method.
4. Delete a quote using the `delete_quote` method.
5. Generate a quote using the `generate_quote` method.

## Testing
Run the tests using `pytest`.
