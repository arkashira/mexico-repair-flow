# Mexico Repair Flow

This project implements a parts inventory and stock management system.

## Features

* Set low-stock alerts for each part
* Visual alert when stock falls below threshold
* Optional email notification for low-stock alerts

## Usage

1. Create an instance of the `Inventory` class
2. Add parts to the inventory using the `add_part` method
3. Set thresholds for each part using the `set_threshold` method
4. Check for low stock using the `check_low_stock` method
5. Send notifications using the `send_notification` method
