# ROADMAP.md

## Introduction
The Mexico Repair Flow is a customer information management system designed to streamline repair service operations in Mexico. The goal is to provide a reliable, scalable system for managing customer data, enabling efficient repair workflows and improving customer satisfaction.

## MVP (Minimum Viable Product)
The MVP will focus on the core functionality required to manage customer information and support basic operations. All features marked as **MVP-critical** are essential for launch.
